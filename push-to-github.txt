# คำสั่งสำหรับ Push โปรเจกต์ไปยัง GitHub

## ขั้นตอนที่ 1: ติดตั้ง Git (ถ้ายังไม่มี)
ดาวน์โหลดจาก: https://git-scm.com/download/win
หลังติดตั้งเสร็จ ให้ปิดและเปิด PowerShell ใหม่

## ขั้นตอนที่ 2: ตั้งค่า Git (ครั้งแรกเท่านั้น)
git config --global user.name "d234mkikodo"
git config --global user.email "dream.seat1@outlook.com"

## ขั้นตอนที่ 3: เปิด PowerShell หรือ Git Bash แล้วรันคำสั่งต่อไปนี้

# 1. เข้าไปยังโฟลเดอร์โปรเจกต์
cd c:\project-xt

# 2. เริ่มต้น Git repository
git init

# 3. เพิ่มไฟล์ทั้งหมด
git add .

# 4. Commit การเปลี่ยนแปลง
git commit -m "Update portfolio: Add Azure project, change email to dream.seat1@outlook.com"

# 5. เชื่อมต่อกับ GitHub repository
git remote add origin https://github.com/d234mkikodo/teeruch-portfolio.git

# 6. Push ขึ้น GitHub
git branch -M main
git push -u origin main

## หมายเหตุ:
- ถ้า repository ยังไม่มี ให้สร้างที่ https://github.com/new
- ตั้งชื่อว่า: teeruch-portfolio
- ตั้งเป็น Public repository
- ไม่ต้องเลือก Initialize with README

## ถ้า repository มีอยู่แล้ว:
git remote set-url origin https://github.com/d234mkikodo/teeruch-portfolio.git
git push -u origin main --force

## หลังจาก Push สำเร็จ:
- เข้าไปที่ Settings > Pages ใน GitHub repository
- เลือก Source: Deploy from a branch
- Branch: main, folder: / (root)
- Save แล้วรอสักครู่ เว็บจะอยู่ที่ https://d234mkikodo.github.io/teeruch-portfolio/

## การอัปเดตครั้งต่อไป:
git add .
git commit -m "Update portfolio"
git push
