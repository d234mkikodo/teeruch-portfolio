A sleek and responsive personal portfolio showcasing my skills in IT, cloud deployment, hardware repair, and Thai-English translation. Built with modern tools and deployed for free.

---

## ✨ Features

- Responsive design using **Tailwind CSS**
- Scroll animations powered by **AOS.js**
- Font Awesome icons for GitHub and LinkedIn
- Clean layout with About, Projects, and Contact sections
- Easy to deploy on Netlify or Vercel
